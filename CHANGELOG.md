# Changelog

## 1.2.0

First public release candidate of the standalone custom firmware repository.

Highlights:

- custom dashboard, metrics page, and admin UI
- cookie-based login page with default `admin` / `admin`
- DHCP default network mode for first boot on unknown LANs
- bundled safe, balanced, and experimental profiles
- on-device MCU firmware generation and apply flow
- firmware upload + manifest apply flow
- PMBus input power integration
- optional debug UART readback integration for live Vddr/Vcore/reset count
- support bundle generation
- 48-hour miner history buffer for charting
