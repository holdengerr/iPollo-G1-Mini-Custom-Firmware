# Firmware Image 1.2.0

## Intended Public Image Contents

Firmware line: `1.2.0`

Core UI surface:

- dashboard
- metrics page
- admin page
- login page

Admin/control features:

- cookie-based login
- DHCP default network mode
- pool editing
- cooling controls
- telemetry controls
- service controls
- support bundle download
- firmware upload/apply
- on-device MCU firmware generation

Telemetry/features:

- PMBus input power path when available
- optional debug UART readback integration
- RAM-backed history buffer used by dashboard and metrics pages

Bundled profiles:

- safe: `1872 / 1300 / 1080`
- balanced: `2004 / 1300 / 1080`
- experimental: `2100 / 1480 / 1060`

Recovery model:

- restore safe profile and restart
